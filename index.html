Block Elements

Block elements start each each element on a new line and take up entire width of the page. Examples are code, div, h1, p, ul, ol, header,footer,section, and article.

Inline Elements

Inline Elements do not start on a new line and only take up as much width as they need to. Examples are: a,img,and span.

Inline-Block

This is used as a display property in the CSS file. It allows a block element to act like an inline elements. This only works if you add a width.

Float Property

The Float Property is used to move an element to one side of the page right or left. Float elements must also have a width.

To put an element underneath or outside of floated elements, we can use the CLEAR property to do so
